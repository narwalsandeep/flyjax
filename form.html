JQuery Flyjax Plugin

Version: 1.0.0

Features:

  • Ajax enable any form
  • Inbuild with Noty Plugin link to noty website
  • Support Success and Failure Callback
  • Option to disable Submit Button
  • Auto fallback of original submit button value when ajax call finishes

Requirement:

  • source.js and noty.js (both provided)
  • Form must have the attribute 'flyjax' like below
    <form method="post" flyjax action="foo.php">
  • There must be a 'button' with type="submit"
consider sourcecode of form on right.

Usage:

$(IDENTIFIER).flyjax(options, successCallback, failureCallback);
To use Noty Plugin just include noty.js provided in your page and pass below config option:
$(IDENTIFIER).flyjax({noty:noty}, successCallback, failureCallback);

Sample Config :

{
enableNoty : true,
noty:noty,
notySuccess : "Form was submitted",
notyFailure : "An error occurred",
disabledSubmitButtonText : "Please wait...",
}
check source.js to see their default value

Code for above form:

			
$('form[ajax-form-submit]').flyjax({
	noty:noty
}
function(form, response) {
	if (response.success) {
		showNoty("success", response.message);
	} else {
		showNoty("error", response.message);
	}
}, function(form, response) {
	showNoty("error", "Cannot process request. Please Try Again!");
});
